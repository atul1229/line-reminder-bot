# Product Backlog

## Product Goal

打造一個以 LINE 為核心入口的個人工作管理系統，降低整理工作資訊的成本，讓使用者每天只需要打開 LINE，就能掌握今天的重要工作。

---

# Priority Definition

Priority 說明：

★★★★★ 必須完成（MVP）

★★★★☆ 應完成（重要）

★★★☆☆ 建議完成

★★☆☆☆ 可延後

★☆☆☆☆ 未來規劃

# Product Rules

新增任何功能前，都必須回答：

- 是否符合 Project Vision？
- 是否能節省使用者時間？
- 是否能降低資訊整理成本？
- 是否會成為每天或每週使用的功能？

如果答案大多是否，則不列入近期開發。
